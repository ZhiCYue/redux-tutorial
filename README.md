# redux-tutorial

## 目录说明

名称 | 描述
-|-
demo01 | redux、redux-thunk、redux-saga 的使用方式
demo02 | react-redux 的使用方式
make-redux | 动手实现 redux

## 运行
切换到示例项目的根目录，例如 demo01
```js
yarn install
yarn start
```

## 其他说明
- 使用了create-react-app 脚手架生成工具
- 其中demo01 中使用了 antd，点击按钮会出现css-animation 异常，应该是 andt 的版本问题（待确定）
